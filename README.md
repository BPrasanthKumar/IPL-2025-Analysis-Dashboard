# 🏏 IPL Season 2025 Analysis Dashboard

## 📌 Project Overview
This project provides a comprehensive analysis of the IPL 2025 season using interactive dashboards built with Excel, SQL, Power BI, and Tableau. The dashboard helps analyze team performance, player statistics, batting insights, bowling insights, venue performance, and season highlights.


## 🎯 Objectives
- Analyze overall IPL 2025 performance.
- Identify top-performing batsmen and bowlers.
- Compare team statistics.
- Track venue-wise performance.
- Create interactive visualizations for better decision-making.


## 🛠️ Tools & Technologies
- Microsoft Excel
- MySQL
- Power BI
- Tableau


## 📊 Dashboard Features

### Overview Dashboard
- Total Matches
- Total Teams
- Centuries & Half-Centuries
- Player of the Match Analysis
- Venue-wise Performance

### Batting Analysis
- Leading Run Scorers
- Highest Fours
- Highest Sixes
- Strike Rate Analysis
- Doubles & Singles Analysis

### Bowling Analysis
- Highest Wicket Takers
- Economy Rate Analysis
- Most Dot Balls
- Overs Bowled
- Runs Conceded Analysis

### Season Highlights
- Orange Cap Winner
- Purple Cap Winner
- Most Valuable Player (MVP)
- Highest Individual Score
- Most Fifties
- IPL 2025 Final Match Summary


## 📈 Key Insights
- Sai Sudharsan emerged as the leading run scorer.
- Prasidh Krishna finished as the highest wicket-taker.
- Narendra Modi Stadium recorded the highest Player of the Match awards.
- Royal Challengers Bengaluru won IPL 2025.
